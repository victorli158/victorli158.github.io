Projects

Small Talk

Small Talk is a mobile video streaming application which allows users to connect with strangers and chat about shared interests. The app uses WebRTC to establish peer-to-peer connections for high-performance streaming between users. Other technologies used include Django, React Native, Redis, and websockets. Mobile transition and loading animations are incorporated to offer a polished and professional application appearance.

NearBytes.me

NearBytes is a full-stack web application (React/Redux, Ruby on Rails, PostgreSQL) which allows users to find and review restaurants in San Francisco. Optimal performance is achieved by using a single-page design and by leveraging Cloudinary to cache images and reduce load times and extraneous AJAX requests. In addition, resolution-responsive CSS stylesheets keep rendering consistent across different device resolutions and browsers. The Google Maps API was integrated to show maps and geolocation data in the search results and restaurant detail pages.

KEYboard

KEYboard is a browser game which allows users to play piano freestyle or play programmed songs for a high score. KEYboard was created using vanilla JavaScript with Canvas, HTML5, and CSS3. Vector math was utilized to handle game state updates within the Canvas element. I limited the number of input keys to eight so input would be manageable for players. In order to allow eight keys to produce all the tones for the more advanced songs, I implemented logic to check if a Note object was within a certain range of the "score line" so only the object's associated tone would be produced by button presses. All in all, working on this project was fun for me as it combined several of my interests!
